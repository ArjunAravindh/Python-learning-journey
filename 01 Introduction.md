# 01 Introduction

## What is Python
Python is an interpreted, high-level programming language. This means that the code is interpreted by an interpreter when the program is run, so there is no need to compile the program before running it. Its sintax is simple, conventional, and close to plain English and standard algebraic notation. Python is very interactive and object-oriented so that  it allows the programmers to model the real life objects and build intuitive program structures.

## Why Python?
Python is one of the best things ever because it scales really well. It's powerful enough to drive complex research and enterprise level applications, and easy enough for a beginner to be able to start writing functional code on day one. The language is designed for readability. This means you spend less tie deciphering complex syntax, and more time focusing on the actual logic of your problem.

## What can you do with it? (Application)
Python is very versatile. Some of its top applications include:
- Web Development
- Software Development
- Mathematics
- Machine Learning
- Artificial Intelligence
- Audio and Video Applications
- Gaming Program
- Networking Programs, etc
