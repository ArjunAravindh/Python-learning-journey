# Topic 01: Hello World Program
## Topic Overview:
* To write and run programs in Python, you can create your code as a standalone script. This means putting all the statements you want to execute into one file and saving it with the .py extension (for example, sample.py). 
* To show text or data on the screen, Python uses the print method, which prints whatever is inside the parentheses and quotes.
## Code Explanation:
* The .py format: Saving your file with a .py extension lets the computer know that the file has a Python script. You can run this script by going to its directory in your terminal and using the command python3 hello_world.py.
* The print() method: The print method is used to show information. It takes the text inside the double quotes, in this case, "Hello World!", and displays it exactly as written on your screen.
